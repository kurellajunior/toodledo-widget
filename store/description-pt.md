## Descrição longa (máx. 4000 caracteres)

Um widget enxuto para a tela inicial do Android que mostra suas tarefas pendentes e futuras do Toodledo — sem enrolação.

Funcionalidades:
• Tarefas atrasadas, de hoje e futuras de relance
• Níveis de prioridade codificados por cor (máxima, alta, normal, baixa)
• Ícones para notas e tarefas recorrentes
• Marcar tarefas como concluídas diretamente no widget
• Toque em uma tarefa para abri-la no app Toodledo ou no navegador
• Atalho "Adicionar tarefa" (toque longo no ícone do app)
• Sincronização automática em segundo plano a cada hora
• Atualização manual por toque
• Redimensionável livremente — de 1×1 até tela cheia
• Transparência e tamanho da fonte ajustáveis
• 12 idiomas (gerados automaticamente com Claude)

Nota sobre o design:
Este widget foi feito para telas sem cantos arredondados forçados. Se o seu launcher (ex.: Android 15 padrão) forçar cantos arredondados nos widgets, a aparência pode ficar um pouco "angulosa".
Dica: Launchers como o Nova Launcher mostram o widget sem arredondamentos forçados.

Privacidade:
Este widget não armazena dados de tarefas localmente. Ele se comunica exclusivamente com a API do Toodledo para buscar tarefas e marcá-las como concluídas. Sem rastreamento, sem anúncios, sem compartilhamento de dados.

Aviso:
Este é um projeto independente sem afiliação com o Toodledo. É necessária uma conta Toodledo.

Notas do desenvolvedor:
Criei este widget porque o Toodledo não tem um próprio, mas eu sempre quero ter minhas tarefas à vista.
