## Descripción larga (máx. 4000 caracteres)

Un widget ligero para la pantalla de inicio de Android que muestra tus tareas pendientes y próximas de Toodledo — sin adornos.

Funciones:
• Tareas atrasadas, de hoy y próximas de un vistazo
• Niveles de prioridad codificados por color (máxima, alta, normal, baja)
• Iconos para notas y tareas recurrentes
• Marca tareas como completadas directamente desde el widget
• Toca una tarea para abrirla en la app de Toodledo o en el navegador
• Atajo "Añadir tarea" (mantener pulsado el icono de la app)
• Sincronización automática en segundo plano cada hora
• Actualización manual tocando
• Tamaño libremente ajustable — de 1×1 a pantalla completa
• Transparencia y tamaño de fuente configurables
• 12 idiomas (generados automáticamente con Claude)

Nota sobre el diseño:
Este widget está hecho para pantallas sin esquinas redondeadas forzadas. Si tu launcher (p. ej. Android 15 estándar) fuerza esquinas redondeadas en los widgets, puede verse un poco "anguloso".
Consejo: Launchers como Nova Launcher muestran el widget sin redondeos forzados.

Privacidad:
Este widget no almacena datos de tareas localmente. Se comunica exclusivamente con la API de Toodledo para obtener tus tareas y marcarlas como completadas. Sin rastreo, sin publicidad, sin compartir datos.

Aviso:
Este es un proyecto independiente sin afiliación con Toodledo. Se requiere una cuenta de Toodledo.

Notas del desarrollador:
Creé este widget porque Toodledo no tiene uno propio, pero siempre quiero tener mis tareas a la vista.
El proyecto está disponible como código abierto en GitHub. Sugerencias y comentarios son bienvenidos: https://github.com/kurellajunior/toodledo-widget
