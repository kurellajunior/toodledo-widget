## Подробное описание (макс. 4000 символов)

Лаконичный виджет для домашнего экрана Android, показывающий ваши текущие и предстоящие задачи из Toodledo — без лишнего.

Функции:
• Просроченные, сегодняшние и предстоящие задачи одним взглядом
• Цветовая кодировка приоритетов (наивысший, высокий, обычный, низкий)
• Иконки для заметок и повторяющихся задач
• Отмечайте задачи выполненными прямо из виджета
• Нажмите на задачу, чтобы открыть её в приложении Toodledo или браузере
• Ярлык «Добавить задачу» (долгое нажатие на иконку приложения)
• Автоматическая фоновая синхронизация каждый час
• Ручное обновление нажатием
• Свободное изменение размера — от 1×1 до полного экрана
• Настраиваемая прозрачность и размер шрифта
• 12 языков (автоматически сгенерировано с помощью Claude)

Примечание о дизайне:
Этот виджет создан для отображения без навязанных скруглённых углов. Если ваш лаунчер (например, стандартный Android 15) принудительно скругляет углы виджетов, внешний вид может быть немного «угловатым».
Совет: Лаунчеры вроде Nova Launcher показывают виджет без принудительного скругления.

Конфиденциальность:
Этот виджет не хранит данные задач локально. Он взаимодействует исключительно с API Toodledo для получения задач и отметки их выполнения. Без отслеживания, без рекламы, без передачи данных.

Отказ от ответственности:
Это независимый проект, не связанный с Toodledo. Требуется учётная запись Toodledo.
